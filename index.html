Проверка автомобиля по VIN

Проверка также возможна по VIN ( ВИН ) - уникальному идентификационному номеру авто. Он содержит информацию о производителе, характеристиках транспортного средства. Его называют также VIN ( ВИН ) кодом, он стоит из семнадцати символов (латинских букв, арабских цифр). Это уникальный номер автомобиля, состоящий из 17 символов, разделенных на 3 части:

  • Всемирный индекс изготовителя (3 символа).
  • Описательная часть (6 символов).
  • Отличительная часть (8 символов).

Общепринятое расположение VIN ( ВИН ): под ветровым стеклом, слева на приборной панели, также на водительской двери. Указанный уникальный идентификационный номер автомобиля - VIN ( ВИН ) применяется для всех авто. Не имеют VIN ( ВИН ) только автомобили из Японии.

VIN в автомобиле

  • на раме передней двери
  • на щитке у ветрового стекла
  • на двигателе
  • на теплоизоляционной перегородке
  • на внутреннем своде левого колеса
  • на руле / рулевой колонке
  • на кронштейне радиатора

VIN в документах

  • в паспорте ТС
  • свидетельстве о регистрации ТС
  • гарантийном талоне / талоне ТО
  • в страховом полисе ОСАГО

Где находится СТС?

Номер свидетельства о регистрации автомобиля продублирован на лицевой (в самом низу) и оборотной стороне документа (вверху над информацией о собственнике). Номер СТС на обеих сторонах выделен красными чернилами
